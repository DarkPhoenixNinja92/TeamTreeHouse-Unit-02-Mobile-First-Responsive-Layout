Portfolio

Project Portfolio

This project shows a rudimentary portfolio where I display my past projects to interested parties.

Number Guessing Game

This project shows a game which generates a random number and tasks the user to guess what number was chosen with feedback based on how close the guess was.

Circle Clicker Game

This project shows a game which generates random circular shapes and then measures how long it takes the user to click on the shape with the speed of arrival for the next shape and the color of the shape being randomly generated.

Palindrome Checker

This project shows an application called a palindrome checker. It takes input from a user and determines if the text entered is the same forward and backwards and prints a line of text based on the outcome.

Random Quote Generator

This project randomly displays quotes upon request. It has a precreated set of quotes stored and will pick one at random and print the quote with the author's name when the user presses the relevant button.

Travel Blog

This project shows a multi-page website which includes travel tips, a blog, personal information and a contact form.
